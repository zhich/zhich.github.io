Java

  • Java 编程思想（第 4 版）「阅读中」

    这算是一本 Java 必读书籍了，有一定基础后再看会好一些，可以放到桌边当工具书。

  • Effective java 中文版（第 2 版）「未阅读」

    这本介绍了 Java 中经常使用但容易被忽略的一些规则，适合对 Java 已经有一定基础的同学。

  • Java 并发编程实战「未阅读」

    多线程并发方面不可多得的好书。

  • 深入理解 Java 虚拟机（第 2 版）「阅读中」

    了解 Java 虚拟机的必读书籍。
